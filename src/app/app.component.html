Featured Components.

Some of the components of the lib

Modal

Modal Component with many features including drag and multi window. The Modal Component is fixed on window, allowing the user move through the window space.

Button

The buttons have many properties to be used, and custom behavior can be attached. It's intelligence when used with Dialog and Form, creating own actions to return for them.

DropdownList

DropdownList Component is drop list that gives the user choose options.It has many features such as filter by specific key of the object and sort by categories

Schedule

Displays appointments across the timetable and allows a user to quickly change between different timetable views using the View Selector and navigate using the Date Navigator.

DataTable

As the name describes, represents massive grid data, with nice features such as a Intelligence LazyLoad Order, Filter and Keyboard Navigation.

Input

Are fields which allow for an input of data, based on a specific and predefined format. With many properties that can be used to different's kind of data or treatment.

Desktop Applications

We developed TrulyUI thinking to fill a gap on open-source Web Technologies that's poor when it's about applications on Desktop. Ex: Electron, App.js and NW.js

Based on Angular

TrulyUI is based on Angular, maintained by world-wide community. By the way, we believe that Angular is the most promising Web Framework.

Free & Open Source

TrulyUI Framework is a 100% free and open source project, licensed under MIT. It will always remain free to use, powered by a massive world-wide community.
